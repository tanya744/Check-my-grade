# CheckMyGrade (DATA 200 – Lab 1)

## Overview
CheckMyGrade is a console-based Python application built using Object-Oriented Programming (OOP) principles.  
It evaluates and manages student grades for different courses.

## Features
- Add / Delete / Modify student, course, professor, and grade records  
- CSV-based data storage  
- Search and sort functionality with timing  
- Calculate average and median scores by course  
- Password encryption and decryption for login  
- Unit testing with 1000+ records for performance measurement  

## Files
- `checkmygrade.py` – Main application  
- `test_checkmygrade.py` – Unit tests  
- `/data/` – CSV files generated when running the program  

## How to Run
```bash
python checkmygrade.py
